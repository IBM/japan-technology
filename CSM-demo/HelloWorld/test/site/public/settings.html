This Example Health patient UI runs as a microservice on Kubernetes. The app was designed to be able to integrate with different kinds of back ends to demonstrate how flexible and versatile microservices can be in a few real world situations.
By default the app just displays preformatted canned data responses in test mode. You can choose to integrate with either a cobol on Z deployment, or a Java Liberty on OpenShift deployment. These integrations are described in related code patterns.
