# IBM Bob ワークショップへようこそ 🎉

IBM Bobを使って、実用的なFAQサイトを一から作成する体験型ワークショップです。プログラミング経験は不要！AI駆動の開発アシスタント「IBM Bob」が、あなたの開発をサポートします。

## 🎯 ワークショップ概要

このワークショップでは、IBM Bobの主要機能を実際に使いながら、FAQサイトの開発プロセスを体験します。

### 学べること

- ✅ IBM Bobの基本的な使い方と画面操作
- ✅ 要件定義書からの自動実装（Orchestratorモード）
- ✅ 既存コードの読み込みと機能追加（Codeモード）
- ✅ AIによるコードレビューと品質向上
- ✅ 開発効率を劇的に向上させるワークフロー

### 対象者

- 💻 **開発者** - 初級〜中級レベル
- 📊 **プロジェクトマネージャー** - 開発プロセスの理解を深めたい方
- 🚀 **エンジニア** - 開発効率化に関心のある方
- 🔍 **IBM Bob導入検討者** - 実際の使用感を体験したい方

!!! note "プログラミング経験不要"
    このワークショップは、プログラミング初心者でも安心して参加できます。IBM Bobが開発をサポートするため、コーディングの詳細な知識は必要ありません。

## ⏱️ 所要時間

**合計**: 約45-65分（最小構成: LAB1のみで約25-40分）

| セクション | 内容 | 想定時間 |
|-----------|------|---------|
| **準備と設定** | 環境構築とファイルのダウンロード | 5-10分 |
| **IBM Bobの画面構成** | 基本操作とインターフェースの理解 | 5-10分 |
| **LAB1: 初期開発** | 要件定義書からFAQサイトを作成 | 15-20分 |
| **LAB2: 機能開発** | 既存コードに追加機能を実装 | 10-15分 |
| **LAB3: セキュリティ強化** | コードレビューと品質向上 | 10分 |

!!! tip "環境準備のヒント"

## 📚 ワークショップの構成

このワークショップは3つのセクションで構成されています：

### 1️⃣ はじめに

IBM Bobの基礎を学び、開発環境を整えます。

- **[事前準備とセットアップ](https://r-nakayamasan.github.io/bob-workshop-draft/00_introduction/)** - ワークショップの概要と必須環境
- **[IBM Bobの紹介](https://r-nakayamasan.github.io/bob-workshop-draft/01_preparation_and_setup/)** - 環境構築とファイルのダウンロード
- **[Bobのインターフェース](https://r-nakayamasan.github.io/bob-workshop-draft/02_bob_interface/)** - 画面構成と基本操作

### 2️⃣ ハンズオン

実際にIBM Bobを使って、FAQサイトを開発します。

#### Lab 1: 要件定義書からの実装 ⭐⭐
**Orchestratorモードを使用**

要件定義書を元に、IBM BobがFAQサイトの基礎部分を自動生成します。HTML、CSS、JavaScriptファイルの作成から、デザインの実装まで、一連の開発プロセスを体験できます。

👉 [Lab 1を始める](https://r-nakayamasan.github.io/bob-workshop-draft/03_lab1/)

#### Lab 2: 既存コードの読み込みと機能追加 ⭐⭐⭐
**Codeモードを使用**

Lab 1で作成したコードを読み込み、新しい機能を追加します。既存プロジェクトへの機能追加という、実務でよくあるシナリオを体験できます。

👉 [Lab 2を始める](https://r-nakayamasan.github.io/bob-workshop-draft/04_lab2/)

#### Lab 3: コードレビュー ⭐
**/reviewコマンドを使用**

IBM Bobの強力なコードレビュー機能を使って、セキュリティやパフォーマンスの問題を自動検出します。AIによる品質向上のプロセスを学びます。

👉 [Lab 3を始める](https://r-nakayamasan.github.io/bob-workshop-draft/05_lab3/)

### 3️⃣ まとめ

- **[まとめと次のステップ](https://r-nakayamasan.github.io/bob-workshop-draft/06_summary/)** - 学んだことの振り返りと、さらなる活用方法
