Your task is to create a Web UI prototype for a credit card management application. 

Since this is a prototype, you will hardcode several components of the application. 

Programming language:
1. Python is used for the implementation of the Web UI tier. 
2. Use streamlit API for implemenation.
3. You may use SQLite to store transactions. 

Web UI tier implementation details:
1. The UI is a mockup of a typical consumer credit card management UI.
2. The UI should show a list mocked up consumer transactions: 15-20 transactions. Subscriptions categories should include: retail, subscription, travel, groceries, gas.
3. The UI should also show 1-2 graphs representing spending by category.
4. The UI should contain a chat interface that will allow a user to ask questions about transactions.
5. The "Send" button of the UI will invoke the route_to_wxo_streaming() method in wxo_agent_adapter.py. 
    Pay special attention to this task. The Python module name is wxo_agent_adapter.py. The method name is route_to_wxo_streaming()
6. Make sure to correctly import Python modules. Pay attention to folder structure when generating import code.
