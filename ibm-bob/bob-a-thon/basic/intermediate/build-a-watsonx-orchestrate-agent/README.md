# Bob-a-thon: IBM Bob とwatsonx Orchestrate でAI エージェントを実装してみよう！
## 開発者向けハンズオン
* 最終更新日: 2026/4/8
* Author: Mika Funato (IBM Japan)

## 📋 本ハンズオン概要

このハンズオンでは、IBM Bob と watsonx Orchestrate を使用してAI エージェントを実装する実践的な演習を行います。
`Part1` ではマルチエージェント構成のクレジットカード管理エージェントを作成し、`Part2` では Web UI を設計・カスタマイズします。

**所要時間**: 30分〜45分

### 🎯 目的

- IBM Bob を使ったハンズオンの全体像を把握する
- `Part1` と `Part2` の学習目的と成果物を理解する
- クレジットカード管理ユースケースを通じて、watsonx Orchestrate のエージェント構成を学ぶ
- Bob を使った UI 設計・実装の進め方を理解する

### 💼 状況の想定

あなたは、IBM Bob と watsonx Orchestrate を活用して業務アプリケーションを構築する開発者です。  
クレジットカードに関する問い合わせを扱うエージェント群を構築し、その後、ユーザー向けのカスタム UI を設計・実装します。

# 2. Part2: Bob を活用してカスタム UI を作成する

**手順書**: [`Part2_custome_ui.md`](Part2_custome_ui.md)  
**動画**: https://ibm.box.com/s/rbex7hyvbeifvcw41gr7jj9nl4iyp5lv

## 🎯 Part2 の目的

Web UI の設計を IBM Bob に指示し、ダッシュボードや取引管理画面をカスタマイズする流れを体験します。

## 🧩 主な機能

### 📊 ダッシュボード
- **概要統計**: 総支出、取引件数、平均取引額、今月の支出を表示
- **カテゴリ別支出**: 円グラフで支出の内訳を可視化
- **時系列分析**: 折れ線グラフで支出トレンドを表示

### 📋 取引管理
- 15〜20件のモックトランザクションデータ
- カテゴリ別フィルタリング機能
- ソート可能なデータテーブル
- 以下のカテゴリをサポート:
  - 小売 (`retail`)
  - サブスクリプション (`subscription`)
  - 旅行 (`travel`)
  - 食料品 (`groceries`)
  - ガソリン (`gas`)

## ✅ Part2 で学べること

- Bob を活用した UI 要件整理の進め方
- モックデータを用いた画面設計の考え方
- ダッシュボード UI の典型的な構成
- エージェント連携を前提としたフロントエンド実装の流れ
