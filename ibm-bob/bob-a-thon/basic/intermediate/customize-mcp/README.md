## ハンズオン手順

###パート1：リモートMCPサーバー
このセクションでは、リモート検索用のMCPサーバーを追加します。普段使用している他のサーバーを追加しても構いません。
1. BobにWeb検索が可能かどうか尋ねます。
2. 検索ツールを追加するための推奨事項を尋ねます。
3. 推奨された方法のいずれかを選択します。テストでは、TavilyのリモートMCPサーバーを使用しました。
4. ドキュメントの手順に従ってサーバーを設定します。
トラブルシューティングのヒント：
• リモートMCP設定に関する既知の問題について、担当インストラクターに確認してください。
• MCPJamなど、別のツールを使ってMCPサーバーの接続をテストしてください。


## パート2：ローカルMCPサーバー
このセクションでは、ローカルMCPサーバーを追加します。普段使用している他のサーバーを追加しても構いません。
1. BobのマーケットプレイスからMCPサーバーを追加します（例：Git系のMCPサーバー）。
2. Bobに対して、いずれかのツールを呼び出すようプロンプトを送ります。
