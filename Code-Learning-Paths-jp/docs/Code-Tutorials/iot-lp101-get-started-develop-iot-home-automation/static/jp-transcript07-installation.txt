Raspberry Pi は、DHT 11 温湿度センサーと PIR センサーが取り付けられた状態になっています。この Raspberry Pi はアクリル・ケースの中に入れてあります。このケースによって Raspberry Pi 上部の回路を保護し、筐体内で回路が短絡したりしないようにします。

筐体には、小さなプラスチック製の食品容器を使います。これが、容器の本体部分です。コンテナーの角のところに、ポケット・ナイフで小さな穴を切り抜いて、ケーブルを容器の中に通せるようにしています。すでにマイクロ USB コードを通してあります。このコードで、Raspberry Pi に電力を供給します。LED ストリップです。この LED ストリップの電源ケーブルで、このケーブルを 12V の電源アダプターに接続することになります。LED ストリップ・ワイヤーは容器の中に通してあって、リレー・モジュールに接続されています。リレー・モジュールの上にはネジ端子があります。この端のところです。LED ストリップにつながっているワイヤーを、これらのネジ端子に接続してあります。よく見ると、ノーマル・オープンのマークが付いていることがわかります。

リレー上にはピンも取り付けられているので、これらのピンを使ってリレーを Raspberry Pi に接続します。それでは、接続しましょう。それにはジャンパー・ワイヤーが必要です。黒のジャンパー・ワイヤーから始めます。アース用のワイヤーです。これをリレー・モジュール上のネガティブ・ピンに差し込みます。リレー・モジュールに接続したら、このワイヤーを Raspberry Pi GPIO ヘッダー上のアース・ピンに接続します。こんな感じです。

次は、5V ワイヤーをリレー・モジュールに接続しなければなりませんが、このように、Raspberry Pi 上にある両方の 5V ピンは使用済みになっています。そこで、とても小さなブレッドボードを使用して、これらのワイヤーを 1 つにまとめます。はんだ付けするという方法もありますが、私はこの小さなブレッドボードを使います。それには追加のジャンパー・ワイヤーが必要になります。そこで、3 本のジャンパー・ワイヤーを追加で用意しました。一方の終端にピンが付いているものです。この 3 本のジャンパー・ワイヤーをブレッドボードの端のレールに差し込みます。これらのジャンパー・ワイヤーから最初の 1 本を使い、Raspberry Pi に接続します。Raspberry Pi 上のすでに使われている 5V ピンのうちの 1 つと差し替えればよいだけです。ブレッドボードにつながっているジャンパー・ワイヤーを Raspberry Pi に接続します。次に、PIR センサーに接続されているジャンパー・ワイヤーをブレッドボードに接続します。ブレッドボード上の中央のジャンパー・ワイヤーには終端にピンが付いているので、そのピンにそのままジャンパー・ワイヤーを差し込むことができます。あるいは、PIR センサーのジャンパー・ワイヤーを交換して、1 本のジャンパー・ワイヤーだけを使うという方法もあります。最後のこのジャンパー・ワイヤーは、リレー・モジュール上の 5V ピンに接続します。この中央にあるピンです。リレー・モジュールには、シグナル・ピン、つまりデータ・ピンもあります。さらにもう 1 本のジャンパー・ワイヤーを使います。これを Raspberry Pi の GPIO ヘッダー上のピン 13 に接続してから、リレー上のシグナル・ピンに接続します。

これで、リレーが Raspberry Pi に接続されました。後は、すべてのコンポーネントをボックスの中に詰め込めばよいだけです。プラスチック製のカバーはすでに PIR センサーから外してあります。ボックスの蓋に小さな穴を切り抜き、その穴に (PIR センサーの) プラスチック製カバーをはめ込んで、ゴムグロメットで固定しました。そこに、PIR センサーを取り付けます。このように、蓋の裏側に取り付けます。蓋に固定するには、両面テープを使います。この小さいテープを蓋の裏に貼ります。そしてセンサーを直接テープの上に貼り付けます。このように、PIR センサーはプラスチック製カバーの背面に配置します。

他のすべてのコンポーネントもボックスの中に収めなければなりません。少々窮屈になりそうです。Raspberry Pi を最初に入れます。ボックスの底にくるようにします。マイクロ USB 電源ケーブルを Raspberry Pi に接続する必要があります。(マイクロ USB 電源ケーブルを) Raspberry Pi の上部にあるマイクロ USB ポートに接続します。こんな感じです。あとは、他のすべてのパーツをボックスの中に詰め込みます。Raspberry Pi とリレー・モジュールがボックスの中に収まった後は、DHT 11 温湿度センサーをボックスから取り出す必要があります。ボックス内に取り付けられていたのでは、正確な値を測定できないからです。ジャンパー・ワイヤーをセンサーから外して、ボックスの横の穴を通して外側に出します。これで、ジャンパー・ワイヤーをボックスの外側にあるセンサーに接続できます。ワイ
ヤーをセンサーに再び接続して、緑のワイヤーはシグナル・ピンにつながっていて、赤のワイヤーは 5V ピンに、黒のワイヤーはネガティブ、つまりアース・ピンにつながっていることを確認します。

後は、これらのワイヤーをすべてボックス内に詰め込めばよいだけです。少々窮屈ですが、最終的にボックス内に収まったら、蓋を戻してボックスを閉めます。ボックスが開かないようにするため、そしてセンサーをボックスの側面に維持するために、ケーブル・タイを使用します。センサーを束ねるようにケーブル・タイを蓋までぐるりと回して締めて、すべてのコンポーネントがボックス内に閉じ込められるようにします。こんな感じです。

これが私のとても単純なホーム・オートメーション・システムです。玄関のドアに正面テープを使って取り付けられる状態になっています。センサーをトリガーすると、ライトが点灯します。それと同時に、iPhone にも通知が送信されます。
