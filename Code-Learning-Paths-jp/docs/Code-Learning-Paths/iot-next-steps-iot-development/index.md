## 概要

「モノのインターネット」は、単なる技術トレンドではなく、技術革命です。身近なモノにセンサーを埋め込み、それらを接続し、収集したデータに分析を適用することで、身の回りのモノがより賢くなります。  この技術革命の一翼を担うためには、より複雑なIoTソリューションのアーキテクチャ、デザイン、開発のベストプラクティスを探求する必要があります。

### 前提知識

このラーニングパスを修了する前に、[Getting started with IoT development](/learningpaths/iot-getting-started-iot-development/)のラーニングパスを修了しておく必要があります。

### スキルレベル

中級者です。

### 完成までの概算時間

約5時間

## 学習目標

このラーニングパスを修了すると、以下のことができるようになります。

* 標準的なIoTアーキテクチャをIoTプロジェクトに適用する方法を理解する。
* IoTソリューションにおける最悪のセキュリティ問題のいくつかを回避する
* IoTデバイスを最適に管理する方法を知ることができます。
* IoTデータを保存し、IoTデータを分析し、IoTデータに基づいて行動するためのルールを適用します。
* 大規模IoTプロジェクトの開始方法を理解する
* Python、MQTTとUDP、IBM Watson IoT Platformを使用するIoTデバイスとIoTアプリケーションを開発します。
