最も効果的なホーム・オートメーション・システムとは、時として、実に単純な問題を解決することを目的としたシステムです。

今日は、とても単純なシステムを構築します。

このシステムは基本的なセンサーを統合したものです。センサーのうちの 1 つは、フロント・ドアの前に人が立つとトリガーして、自動的に LED ストリップ・ライトを点灯させることができます。ですから夜中に帰宅したときに、暗闇の中でごそごそ手探りする必要がなくなります。

これから、この非常に基本的なシステムを Raspberry Pi と Node-RED を使って構築するので、その手順をご覧ください。
