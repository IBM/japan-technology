# AI Assistant Builderを使った AIアシスタントの作成

* 最終更新日: 2025/11/10
* このハンズオンは、watsonx OrchestrateのAI Assistant Builderを使って、社内のヘルプデスクで利用することを想定して、簡単なチャットボット作成を体験します。
* watsonx OrchestrateのSaaS環境だけで作成しますので、外部のシステムや文書との連携は含みません。

##

## 1. Wi-Fiパスワードの案内
下記のような問い合わせに対して、ユーザーが現在いるオフィスを確認し、該当のWi-Fiパスワードを案内します。
* Wi-Fiが繋がらない
* Wi-Fiパスワード
* 無線LANのパスワード
* wifi password
* 社内ネットワーク つながらない

## 2. PC故障時の案内
下記のような問い合わせに対して、故障中のOSとメーカー条件に応じてサポート窓口を案内します。
* PC故障
* PCが壊れた
* パソコンが起動しない
* ブルースクリーン
* Macが動かない
