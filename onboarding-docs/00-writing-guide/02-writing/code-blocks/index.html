コードブロック

コードを効果的に表示する方法
最終更新: 2026/2/9

コードブロックは、コマンドや設定ファイルを明確に表示するための重要な要素です。

基本的なコードブロック

インラインコード

文中でコードを表示する場合は、バッククォート`で囲みます。

`npm run dev`コマンドを実行します。

プレビュー： npm run devコマンドを実行します。

コードブロック

複数行のコードは、3つのバッククォート```で囲みます。

```bash
npm install
npm run dev
```

プレビュー：

npm install
npm run dev

言語指定

コードブロックの最初に言語名を指定すると、シンタックスハイライトが適用されます。

```typescript
export default defineNuxtConfig({
  modules: ['@nuxt/ui-pro']
})
```

プレビュー：

export default defineNuxtConfig({
  modules: ['@nuxt/ui-pro']
})

対応言語

主な対応言語：

  • bash / sh - シェルコマンド
  • javascript / js - JavaScript
  • typescript / ts - TypeScript
  • python / py - Python
  • yaml / yml - YAML
  • json - JSON
  • markdown / md - Markdown
  • html - HTML
  • css - CSS
  • sql - SQL

ファイル名付きコードブロック

言語名の後に[ファイル名]を追加すると、コードブロック上部にファイル名が表示されます。

```yaml [.product.yml]
title: "製品名"
description: "製品の説明"
group: "watsonx"
```

プレビュー：

.product.yml
title: "製品名"
description: "製品の説明"
group: "watsonx"
ファイル名を指定すると、拡張子に応じたアイコンが自動的に表示されます。

CodeGroup（コードグループ）

複数のコードブロックをタブで切り替えます。パッケージマネージャーごとのコマンドなどに便利です。

基本的な使い方

::code-group
```bash [npm]
npm install
```

```bash [yarn]
yarn install
```

```bash [pnpm]
pnpm install
```
::

プレビュー

npm install

yarn install

pnpm install

CodeTree（コードツリー）

ファイルツリー表示でコードブロックを表示します。プロジェクト構造の説明に最適です。

基本的な使い方

::code-tree{default-value="app/app.config.ts"}
```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@nuxt/ui-pro']
})
```

```ts [app/app.config.ts]
export default defineAppConfig({
  ui: {
    colors: {
      primary: 'blue'
    }
  }
})
```

```vue [app/app.vue]
<template>
  <UApp>
    <NuxtPage />
  </UApp>
</template>
```
::

プレビュー

app/app.config.ts
export default defineAppConfig({
  ui: {
    colors: {
      primary: 'blue'
    }
  }
})
default-valueプロパティで、デフォルトで表示するファイルを指定できます。

CodePreview（コードプレビュー）

コードとその出力結果を並べて表示します。

基本的な使い方

::code-preview
`inline code`

#code
```markdown
`inline code`
```
::

プレビュー

inline code

`inline code`
#codeスロットにコードを記述し、デフォルトスロットにプレビューを記述します。

CodeCollapse（コード折りたたみ）

長いコードブロックを折りたたんで表示します。

基本的な使い方

::code-collapse
```css [main.css]
@import "tailwindcss";
@import "@nuxt/ui-pro";

@theme {
  --font-sans: 'Public Sans', sans-serif;
  --breakpoint-3xl: 1920px;
  
  --color-green-50: #EFFDF5;
  --color-green-100: #D9FBE8;
  --color-green-200: #B3F5D1;
  /* ... 長いコード ... */
}
```
::

プレビュー

main.css
@import "tailwindcss";
@import "@nuxt/ui-pro";

@theme {
  --font-sans: 'Public Sans', sans-serif;
  --breakpoint-3xl: 1920px;
  
  --color-green-50: #EFFDF5;
  --color-green-100: #D9FBE8;
  --color-green-200: #B3F5D1;
  --color-green-300: #75EDAE;
  --color-green-400: #00DC82;
}
