ワークフローとリファレンス

クイックリファレンス

全設定パラメータとよく使うコンポーネントの早見表
最終更新: 2026/2/9

このページは、設定パラメータとコンポーネントのクイックリファレンスです。

設定ファイル

.product.yml

製品の基本情報を定義します。

title: "製品名"                    # 必須：製品の表示名
description: "製品の説明"          # 必須：製品の簡潔な説明
group: "グループ名"                # 必須：所属グループ
image: "アイコンまたは画像パス"    # 必須：i-icon-name または /path/to/image.png
order: 1                          # 必須：表示順序
forQuote: true                    # オプション：引用専用（本番非表示）
dev: true                         # オプション：開発中（本番非表示）

詳細： 製品の作成

.groups.yml

グループの定義と順序を設定します。

groups:
  - name: "グループ名"             # 必須：グループの表示名
    order: 1                      # 必須：表示順序
    dev: true                     # オプション：開発中（本番非表示）

詳細： 製品の作成

.navigation.yml

セクションの情報を定義します。

title: "セクション名"              # 必須：セクションの表示名
icon: "i-carbon-icon-name"       # オプション：アイコン
order: 1                         # オプション：表示順序

詳細： セクションとページの作成

Frontmatter

ページのFrontmatter

---
title: "ページタイトル"            # 必須：ページのタイトル

order: 1                         # オプション：表示順序
dev: true                        # オプション：開発中
---

カスタムコンポーネント

:toc

製品ガイドのセクション一覧を自動生成します。

:toc

詳細： トップページの作成

:page-comp

他のページのコンテンツを参照します。

:page-comp{page="/パス"}
:page-comp{page="/パス" extends=true}

パラメータ：

  • page（必須）：参照するページのパス
  • extends（オプション）：trueでメタデータを継承

詳細： ページ参照機能

:video

動画を埋め込みます。

:video{autoplay controls loop src="動画のURL"}

Markdown基本記法

要素記法
見出し## H2, ### H3, #### H4
太字**text**
斜体*text*
打ち消し~~text~~
リンク[text](url)
サイト内リンク[text](/path)
画像![alt](/path/image.png)
インラインコード`code`
箇条書き- item
番号付きリスト1. item
引用> text
表| col1 | col2 |

Gitコマンド

基本操作

ステージング：

git add .

コミット：

git commit -m "メッセージ"

プッシュ：

git push

直前のコミット取り消し：

git reset --soft HEAD^

ブランチ操作

ブランチ作成：

git checkout -b doc/branch-name

ブランチ切り替え：

git checkout branch-name

直前のブランチに戻る：

git checkout -

developの最新化

git checkout develop
git pull
git checkout -
git merge develop

詳細： Git操作

ビルドコマンド

開発サーバー：

npm run dev

IBM/japan-technology用デプロイ：

npm run build:jpn-tech

customer-success-japan用デプロイ：

npm run build:ghp-int

PDF生成：

npm run build:pdf

アイコンの探し方

  1. Icones.js.org - Carbonにアクセス
  2. 検索ボックスでアイコンを検索
  3. アイコン名をコピー
  4. i-プレフィックスを付けて使用

使用するコレクション：

  • i-carbon-*：IBM Carbonアイコン
