高度な機能

Instanaモニタリング等の高度な機能
最終更新: 2026/2/9

このページでは、製品ガイド作成の高度な機能について解説します。

Instanaモニタリング

サイトには、Instana EUM（End User Monitoring）が統合されています。

機能

  • ページビューの追跡
  • ユーザーセッションの追跡
  • パフォーマンスメトリクスの収集

アクセス方法

からダッシュボードにアクセス可能です。

VS Code設定

ワークスペースには、画像の自動配置設定が含まれています。

画像の自動配置

.vscode/settings.json（gitignoreで無視）に以下を設定すると、画像が自動的にpublic/内に配置されます。

.vscode/settings.json
{
  "markdown.copyFiles.destination": {
    "**/*.md": "/public/製品名/${fileName}"
  }
}
執筆中の製品名に合わせて設定を作成してください。

PDF生成

ドキュメントをPDF形式で出力できます。

コマンド

npm run build:pdf

機能

  • 製品ごとにPDFを生成
  • 目次の自動生成
  • ページ番号の追加
  • 印刷最適化されたスタイル
PDF生成には、開発サーバーが一時的に起動されます。
