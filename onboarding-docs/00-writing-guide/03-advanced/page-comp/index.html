高度な機能

ページ参照機能

共通コンテンツを再利用するpage-compコンポーネントの使い方
最終更新: 2026/2/9

:page-compコンポーネントを使用すると、他のページのコンテンツを参照して表示できます。共通手順の再利用に便利です。

基本的な使い方

インライン記法

:page-comp{page="/共通ページのパス"}

ブロック記法

::page-comp{page="/共通ページのパス"}
::
どちらの記法でも同じ結果になります。好みに応じて使い分けてください。

パラメータ

page
string required
参照するページのパスを指定します。
:page-comp{page="/cloud-instance/01_ibmcloud_login"}
パスの指定方法：
  • /で始まるcontentディレクトリ内の絶対パス
  • .md拡張子は不要
パスは小文字に正規化されます。大文字を含むパスは小文字に変換されて処理されます。
extends
boolean
trueに設定すると、参照元ページのtitleとdescriptionを現在のページに継承します。
:page-comp{page="/cloud-instance/01_ibmcloud_login" extends=true}
用途：
  • 共通ページの内容をそのまま使用し、タイトルと説明も引き継ぎたい場合

使用例

基本的な参照

共通ページ：

content/cloud-instance/01_ibmcloud_login.md
---
title: "IBM Cloudへのログイン"
description: "IBM Cloudにログインする手順を説明します"
---

## ログイン手順

1. [IBM Cloud](https://cloud.ibm.com/)にアクセス
2. IBMidでログイン
3. ダッシュボードが表示されることを確認

製品ページから参照：

content/watsonx-ai/01_instance/01_ibmcloud_login.md
---
title: "IBM Cloudへのログイン"
description: "watsonx.aiのインスタンスを作成するため、IBM Cloudにログインします"
---

:page-comp{page="/cloud-instance/01_ibmcloud_login"}

extends機能の使用

共通ページのタイトルと説明をそのまま使用する場合：

content/watsonx-ai/01_instance/01_ibmcloud_login.md
---
# titleとdescriptionは省略可能（共通ページから継承）
---

:page-comp{page="/cloud-instance/01_ibmcloud_login" extends=true}
extends=trueを使用すると、現在のページにtitleとdescriptionがない場合、参照元ページのものが使用されます。

共通ページの作成

引用専用ページの設定

引用専用のページを作成し、forQuote: trueを設定します。

content/cloud-instance/.product.yml
title: "IBM Cloud 共通手順"
description: "IBM Cloud インスタンス作成の共通手順"
group: "共通"
image: "i-carbon-ibm-cloud"
forQuote: true
order: 999

トップページの表示：

  • 本番環境：製品一覧に表示されない
  • 開発環境：一覧に表示され、QUOTEバッジが付く

開発中ページの警告

参照元ページの製品が開発中（dev: true）の場合、警告が表示されます。

開発中製品の設定

content/new-product/.product.yml
title: "新製品（開発中）"
description: "現在作成中の製品ガイドです"
group: "watsonx"
dev: true
order: 99

警告の表示

開発環境：

  • 警告ブロック + コンテンツを表示
  • 「開発中の製品のページを参照しています」というメッセージ

本番環境：

  • 警告ブロックのみ表示
  • コンテンツは非表示

目次（TOC）の継承

:page-compで参照したページの見出しも、現在のページの見出しと同様に右側の目次に表示されます。

エラーハンドリング

存在しないページを参照

:page-comp{page="/non-existent-page"}

表示：

ページの読み込みに失敗しました指定されたページが見つからないか、読み込み中にエラーが発生しました。参照パス: /non-existent-page
