セクションとページの作成

製品ガイドのセクション（ディレクトリ）とページ（Markdownファイル）の作成
最終更新: 2026/2/9

製品ガイドは、セクション（ディレクトリ）とページ（Markdownファイル）で構成されます。

ディレクトリ構造の設計

基本的な構造

content/
  製品名/
    .product.yml
    index.md
    
    01_セクション1/
      .navigation.yml
      01_ページ1.md
      02_ページ2.md
    
    02_セクション2/
      .navigation.yml
      01_ページ1.md
      02_ページ2.md

セクションの作成

1. ディレクトリの作成

mkdir content/製品名/01_セクション名

例：

mkdir content/watsonx-ai/01_instance

2. .navigation.ymlの作成

各セクションディレクトリ内に.navigation.ymlを作成します。

基本的な設定
content/製品名/セクション名/.navigation.yml
title: "セクションのタイトル"
icon: "i-carbon-icon-name"
order: 1
パラメータ詳細
title
string required
セクションの表示名です。ナビゲーションメニューと:tocに表示されます。
title: "インスタンスの作成"
icon
string
ナビゲーションメニューに表示されるアイコンです。
icon: "i-carbon-cloud"
アイコンの検索方法については、アイコンについてを参照してください。アイコンが不要な場合は、iconパラメータを省略するか、falseを設定します。
order
number
セクションの表示順序です。数字が小さいほど上に表示されます。
order: 1
orderがない場合は、ディレクトリ名でソートされます。
設定例
content/watsonx-ai/01_instance/.navigation.yml
title: "インスタンスの作成"
icon: "i-carbon-cloud"
order: 1
content/instana/what_is_instana/.navigation.yml
title: "Instanaとは"
order: 2

ページの作成

1. Markdownファイルの作成

セクションディレクトリ内に.mdファイルを作成します。

touch content/製品名/セクション名/01_ページ名.md

例：

touch content/watsonx-ai/01_instance/01_ibmcloud_login.md

2. Frontmatterの設定

各ページの先頭に、YAMLフォーマットでメタデータを記述します。

基本的な設定
content/製品名/セクション名/ページ名.md
---
title: "ページのタイトル"
description: "ページの説明"
---
パラメータ詳細
title
string required
ページのタイトルです。ページ上部、ナビゲーションメニュー、:tocに表示されます。
title: "IBM Cloudへのログイン"
description
string required
ページの説明です。タイトル下のグレーテキストとして表示されます。
description: "IBM Cloudにログインして、インスタンス作成の準備をします"
icon
string
サイドバーに表示されるアイコンです。
icon: "i-carbon-login"
アイコンの検索方法については、アイコンについてを参照してください。
order
number
ページの表示順序です。数字が小さいほど上に表示されます。
order: 1
orderがない場合は、ファイル名でソートされます。
設定例
基本的なページ
content/watsonx-ai/01_instance/01_ibmcloud_login.md
---
title: "IBM Cloudへのログイン"
description: "IBM Cloudにログインして、インスタンス作成の準備をします"
---

## ログイン手順

1. [IBM Cloud](https://cloud.ibm.com/)にアクセス
2. IBMidでログイン
...

表示順序の制御

ページとセクションの表示順序は、以下の優先順位で決定されます：

  1. orderパラメータ
    • .navigation.ymlのorder（セクション）
    • Frontmatterのorder（ページ）
  2. ファイル/ディレクトリ名でソート（orderがない場合）

チェックリスト

セクションとページの作成が完了したら、以下を確認してください：

  • セクションディレクトリを作成した
  • .navigation.ymlを作成し、titleを設定した
  • ページファイル（.md）を作成した
  • 各ページのFrontmatterにtitleとdescriptionを設定した
  • 開発サーバーで以下の項目を確認した
    • トップページの:tocにセクションとページが表示される
    • 左側のナビゲーションメニューにセクションが表示される
    • セクション内にページが表示される
    • 表示順序が正しい
    • アイコンが表示される（設定した場合）
