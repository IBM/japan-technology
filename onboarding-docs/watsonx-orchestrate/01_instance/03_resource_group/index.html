リソースグループの作成

リソースグループの作成方法
最終更新: 2025/8/22
  1. 右上の 管理 → アカウント をクリックする
  2. 左側の リソース・グループ をクリックする
  3. 作成 をクリックする
  4. 名前に RG_XXXXX (名前は任意の文字列) と入力し、作成 をクリックする
