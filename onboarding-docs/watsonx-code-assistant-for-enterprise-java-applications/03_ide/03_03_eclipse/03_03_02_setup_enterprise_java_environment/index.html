エンタープライズJava環境のセットアップ

エンタープライズJava環境のセットアップ方法
最終更新: 2025/8/25
  1. Java および Mavenをインストールする　インストールされていない場合
  • （参考）Java
  • （参考）Maven
  1. Eclipse IDEのメニューからWindow をクリックする
  2. 表示されたリストからPreferences をクリックする
  3. Java -> Installed JREs をクリックする
  4. Javaが追加されていない場合、Javaを追加する
1. **Add** ボタンをクリックする

![](/watsonx-Code-Assistant-for-Enterprise-Java-Applications/03_ide/03_03_eclipse/03_03_02_003.png)

1. 適切なJREタイプを選択し、**Next** ボタンをクリックする
![](/watsonx-Code-Assistant-for-Enterprise-Java-Applications/03_ide/03_03_eclipse/03_03_02_004.png)

1. 適切なJava情報を設定し、**Finish** ボタンをクリックする
![](/watsonx-Code-Assistant-for-Enterprise-Java-Applications/03_ide/03_03_eclipse/03_03_02_005.png)

1. 追加されたJavaを選択し、**Apply** ボタンをクリックする
![](/watsonx-Code-Assistant-for-Enterprise-Java-Applications/03_ide/03_03_eclipse/03_03_02_006.png)

1. **Apply and Close** ボタンをクリックする
![](/watsonx-Code-Assistant-for-Enterprise-Java-Applications/03_ide/03_03_eclipse/03_03_02_007.png)
  1. Path変数にMavenを追加する　※設定されていない場合
    例：
    変数名値
    PathC:\Program Files\apache-maven-3.9.9\bin
