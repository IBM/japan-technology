Eclipse IDEプラグインのインストール

Eclipse IDEプラグインのインストール方法
  1. Eclipse IDEのメニューからHelp をクリックする
  2. 表示されたリストからEclipse Marketplace をクリックする
  3. 表示された入力欄に watsonx Code Assistant for Enterprise Java Applications と入力し、Goボタンをクリックする
  4. プラグイン 一覧に表示された IBM watsonx Code Assistant for Enterprise Java Applications の インストール ボタンをクリックする
  5. Review LicensesでI accept the terms of license agreementsを選択し、Finish ボタンをクリックする
  6. Trust AuthoritiesでAuthority / Update Site に表示されたチェックボックスを選択し、Trust Selected ボタンをクリックする
  7. Software UpdatesでRestart Now ボタンをクリックする

プラグインのセットアップ

  1. Eclipse IDEのメニューからWindow をクリックする
  2. 表示されたリストからShow View -> Otherをクリックする
  3. Show Viewダイアログでwatsonx Code Assistant フォルダを開き、Chat を選択して、Open ボタンをクリックする
  4. Log in with your API Key ボタンをクリックする
  5. 表示された画面のAPI Key に保管しておいたAPIキーを入力し、Apply and Close ボタンをクリックする
  6. 以下の画面が表示されたらプラグインのセットアップは完了
