watsonx.ai にアクセス

watsonx.ai にアクセス
最終更新: 2025/11/14
  1. IBM Cloud にアクセスする
  2. ①左上のハンバーガーメニューから、②リソースリストを選択する
  3. ①AI/機械学習、②watson.ai Studio を選択する
  4. ①ドロップダウンを開き、②IBM watsonx をクリックしてwatsonx.aiを起動する
  5. 以下の画面が表示されたらwatsonx.ai へのアクセスは完了
    ※Cloud Accountを複数持っている場合は勝手に切り替わることがあるため、Accountに間違いがないか確認する
